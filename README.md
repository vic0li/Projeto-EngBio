# brain-tumor-AI
Este projeto apresenta um modelo de Inteligência Artificial desenvolvido para auxiliar na detecção de tumores cerebrais a partir de imagens de ressonância magnética (MRI). Utilizando técnicas de Deep Learning e Visão Computacional, o sistema é capaz de classificar exames em diferentes categorias: glioma, meningioma, tumor de hipófise (pituitária) e ausência de tumor.

O modelo foi treinado com a arquitetura MobileNetV2, aplicando Transfer Learning para aumentar a eficiência e o desempenho da classificação das imagens médicas. Além disso, foi desenvolvida uma interface interativa em Streamlit, permitindo que o usuário realize o upload de imagens e visualize automaticamente os resultados da análise.

A base de dados utilizada no treinamento foi o dataset público “Brain Tumor MRI Dataset”, disponível no Kaggle, contendo milhares de imagens de ressonância magnética organizadas em diferentes classes tumorais.
